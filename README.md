# 🍔 Food Ordering System (Django Project)

## 📌 Project Overview
A full-stack Food Ordering Web Application built using Django.  
Users can register, login, browse food items, add to cart, place orders, and receive order confirmation via email.

---

## 🚀 Features
- User Authentication (Register/Login/Logout)
- Add to Cart functionality
- Order Placement System
- Email Confirmation after successful payment
- Admin Panel for managing food items and orders
- Secure Environment Variables for Email Credentials

---

## 🛠️ Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
- Git & GitHub

---

## ⚙️ How to Run This Project

1. Clone the repository

git clone https://github.com/rishi23200-oss/Food-Ordering-System.git


2. Create virtual environment
python -m venv env

3. Activate virtual environment
env\Scripts\activate

4. Install requirements
pip install -r requirements.txt

5. Run server
python manage.py runserver
